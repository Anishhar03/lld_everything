## Answer Checklist For Every Question

- Functional requirements.
- Non-functional requirements.
- Scale assumptions.
- API design.
- Data model.
- High-level architecture.
- Deep dive into bottleneck.
- Caching strategy.
- Database scaling strategy.
- Consistency model.
- Failure handling.
- Observability.
- Security/rate limiting.
- Tradeoffs.
