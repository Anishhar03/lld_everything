# Design Patterns

Every pattern folder now contains intent, motivation, structure, implementation path, tradeoffs, and interview explanation.

- [Singleton](creational/singleton/README.md) - Ensure a class has only one instance and provide a global access point.
- [Factory Method](creational/factory-method/README.md) - Create objects without exposing concrete creation logic to client code.
- [Abstract Factory](creational/abstract-factory/README.md) - Create families of related objects without depending on concrete classes.
- [Builder](creational/builder/README.md) - Build complex objects step by step with readable construction and validation.
- [Prototype](creational/prototype/README.md) - Create new objects by copying existing objects.
- [Adapter](structural/adapter/README.md) - Convert one interface into another expected by client code.
- [Bridge](structural/bridge/README.md) - Separate abstraction from implementation so both can vary independently.
- [Composite](structural/composite/README.md) - Treat individual objects and groups of objects uniformly.
- [Decorator](structural/decorator/README.md) - Add behavior dynamically by wrapping an object.
- [Facade](structural/facade/README.md) - Provide a simple interface over a complex subsystem.
- [Flyweight](structural/flyweight/README.md) - Share immutable intrinsic state to reduce memory usage.
- [Proxy](structural/proxy/README.md) - Control access to another object.
- [Chain Of Responsibility](behavioral/chain-of-responsibility/README.md) - Pass a request through a chain of handlers until handled.
- [Command](behavioral/command/README.md) - Encapsulate a request as an object.
- [Iterator](behavioral/iterator/README.md) - Traverse a collection without exposing its internal representation.
- [Mediator](behavioral/mediator/README.md) - Centralize communication among many objects.
- [Memento](behavioral/memento/README.md) - Capture and restore object state without exposing internals.
- [Observer](behavioral/observer/README.md) - Notify subscribers when an object changes state.
- [State](behavioral/state/README.md) - Let an object change behavior when its internal state changes.
- [Strategy](behavioral/strategy/README.md) - Select an algorithm at runtime.
- [Template Method](behavioral/template-method/README.md) - Define an algorithm skeleton while subclasses customize steps.
- [Visitor](behavioral/visitor/README.md) - Add operations to object structures without changing element classes.
